# SwordsNSandals

A simple CLI-based Java game. Fight for your freedom as a gladiator!

# Setup

1. cd into the 'SwordsNSandals' directory
2. 'javac SwordsNSandals.java'

# Running

1. 'java SwordsNSandals.java'
2. Enjoy the game!
